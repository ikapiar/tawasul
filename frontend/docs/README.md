# Tawasul Frontend Developer Docs

Welcome! This documentation is the fastest way to contribute to the Tawasul frontend.

If you are new to Angular or not confident with CSS, you can still build features following these guides. We use Angular 20 (standalone components) and Angular Material (M3) for almost all UI, with minimal custom CSS.

Quick links:
- Codebase root for the frontend: `frontend/`
- App source: `frontend/src/app/`
- Layouts: `frontend/src/app/layouts/`
- Features: `frontend/src/app/features/`
- Services: `frontend/src/app/services/`
- Global styles: `frontend/src/styles.css`
- Material theme: `frontend/src/custom-theme.scss`

If in doubt, prefer Angular Material components over custom styling.
